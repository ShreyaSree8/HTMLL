JS stands for JavaScript. It is a text-based, lightweight, cross-platform, and interpreted scripting programming language.This language is very popular for developing web pages.

HTML

'HyperText Markup Language' is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

CSS

'Cascading Style Sheets' is a computer language for laying out and structured web pages (HTML or XML). This language contains coding elements and is composed of these “cascading style sheets” which are equally called CSS files
